Profil Saya

Rahma Nur Istiqomah

Halo! Saya Rahma Nur Istiqomah, seorang siswi jurusan Rekayasa Perangkat Lunak yang bersemangat dalam dunia pengembangan web dan UI/UX. Saat ini, saya menempuh pendidikan di SMK Negeri 2 Balikpapan dan mempelajari berbagai teknologi seperti HTML, CSS, Python, PHP, JavaScript, Laravel, dan MySQL. Saya juga terbiasa menggunakan Visual Studio Code dan Figma sebagai alat pendukung pengembangan.

Saya percaya pada pembuatan solusi yang bersih, efisien, dan berpusat pada pengguna. Selama Praktik Kerja Lapangan (PKL), saya terlibat dalam pengembangan aplikasi Point of Sale (POS), yang memberi saya pengalaman langsung dalam memahami siklus pengembangan perangkat lunak.

Dengan semangat untuk terus belajar dan beradaptasi dengan teknologi terbaru, saya berharap dapat berkontribusi pada proyek-proyek inovatif di masa depan dan menghadirkan solusi digital yang bermanfaat bagi banyak orang.
